## 🐛 If Something Goes Wrong

Don't panic! Most issues are easy to fix:

1. **Check Render logs** (Service → Logs tab)
2. **Check browser console** (F12 key)
3. **Review [DEPLOYMENT_TROUBLESHOOTING.md](DEPLOYMENT_TROUBLESHOOTING.md)**
4. **Verify environment variables** are all correct

99% of issues are one of:
- Wrong database credentials
- Wrong API URL in .env.production
- Missing environment variable
- Git push didn't work

## 💡 Pro Tips

1. **Add a health check endpoint:**
   - Render checks `/api/health` by default
   - Keep it fast to prevent timeouts

2. **Monitor logs regularly:**
   - Set Render to email you on failures

3. **Set up alerts:**
   - Get notified of service issues

4. **Backup your database:**
   - Render includes backups, but always safe!

5. **Use strong JWT secret:**
   - Change from default in environment variables
